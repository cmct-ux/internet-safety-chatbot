# Chatbot An Toàn Internet Cho Trẻ Em

## 1. Mục tiêu
Giúp trẻ em nhận biết nguy hiểm trên Internet và biết cách tự bảo vệ bản thân.

## 2. Đối tượng sử dụng
- Trẻ em
- Phụ huynh
- Giáo viên

## 3. Chức năng
- Tư vấn an toàn Internet
- Nhận diện tình huống nguy hiểm
- Cảnh báo rủi ro

## 4. Công nghệ
- HTML, CSS, JavaScript
- Chatbot rule-based

## 5. Hướng phát triển
- Ứng dụng AI học máy
- Gửi cảnh báo phụ huynh
- Phát triển app điện thoại
